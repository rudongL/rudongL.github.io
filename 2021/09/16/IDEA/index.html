IDEA 快捷键

  • 快捷键
    • 编辑
      • 重写方法 —— ctrl + O
      • 自动修复 —— alt + enter
      • 格式化代码 —— ctrl + alt + L
      • 扩大文本选中范围 —— ctrl + w
      • 缩小选中范围 —— ctrl + shift + w
      • 复制当前行 —— ctrl + d
      • 删除选中行 —— ctrl + y
      • 移动当前行 —— alt + shift + ↑|↓
      • 移动代码块 —— ctrl + alt + ↑|↓
      • 整体左移 —— Shift + Tab
      • 整体右移 —— Tab
      • 展开|折叠方法
        • 当前方法 —— ctrl + + | -
        • 所有方法 —— ctrl + shift + + | -
      • 插入模板代码 —— ctrl + alt + t
      • 撤销上次操作 —— ctrl + shift + delete
      • 标签操作
        • 选中当前标签 —— ctrl + j
        • 选中同类标签 —— ctrl + shift + j
        • 取消选中标签 —— alt + shift + j
      • 全局替换 —— ctrl + shift + r
      • 前进|后退 —— ctrl + alt + ←|→
      • 禅模式 —— Alt+Z
    • 提示 & 补全
      • 查看全部提示 —— ctrl + space 两次
      • 自动补全（; for if等） —— ctrl + shift + enter
      • 查看所有匹配建议 —— ctrl + shift + space
      • 自动补全
        • .
        • tab
      • 查看补全建议 —— ctrl + space
      • 参数提示 —— ctrl + p
      • 查看文档 —— ctrl + q
      • 方法定义 —— ctrl + shift + i
      • 高亮所有变量 —— ctrl + shift + F7
    • 重构
      • 批量重命名 —— shift + F6
      • 抽取变量 —— ctrl + alt + v
      • 抽取方法 —— ctrl + alt + m
      • 查看重构列表 —— ctrl + alt + shift + t
    • 格式化代码
      • 格式化代码 —— ctrl + alt + L
      • 设置格式化的位置 —— ctrl + shift + alt + L
      • 代码缩进 —— ctrl + alt + I
    • 搜索
      • 搜索所有 —— 双击 shift
      • 搜索类 —— ctrl + n
      • 在指定文件夹搜索类 —— ctrl + shift + n
      • 全局变量或方法 —— ctrl + shift + alt + n
      • 切换搜索位置 —— alt + p
      • 查找所有指定字符串 —— ctrl + shift + f 严格匹配 —— alt + w
      • 在文件中查找，f3 下一个 shift f3 上一个 —— ctrl + F
    • 文件结构
      • 打开项目结构 —— ctrl + f12
      • 另一种方式 —— alt + 7
      • 跳到方法声明处，再次查看所有使用；查看方法实现 —— ctrl + B
      • alt + f7 方法使用细节，包括声明和使用，右键 pin tab（锁定标签），shift + esc 隐藏视图 alt + 3 重新查看使用
      • 父方法 —— ctrl + u
      • 接口实现树 —— ctrl + shift+ h
      • 类继承图 —— ctrl + h
      • 切换最近文件 —— ctrl + e
      • 最近访问文件的代码 —— ctrl + shift + e
    • 运行和调试
      • 运行项目 —— ctrl + shift + f10
      • 打断点 —— ctrl + f8
      • 调试方式运行 —— shift + f9
      • 进入方法 —— f7
      • 跳出方法 —— shift + f8
      • 执行方法下一行 —— F8
      • 下一个断点 —— F9
      • 定位到下一个错误位置 —— f2
      • 查看异常信息 —— ctrl + f1
      • [[IDEA01]]
    • Git 相关
      • add – ctrl + alt + a
      • rollback– ctrl + alt + z
